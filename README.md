# Introduction
Automated scripting for RFQs, extracting information from PDF and store them in Railway MySQL database. Lots of cool things to be added in here.

The purpose of this project is to automate many tasks that could automate, increasing efficiency and accuracy by at least 20%, and to make the pricing more competitve. This repository will also display project numbers to indicate what project it is, and how many projects are in here. 

>[!NOTE] 
>This project leverages AI-assisted code optimization to streamline parsing, data cleaning, and automation processes. The logic workflow were developed and refined through iterative testing and validation.
